Informações sobre redes

Servidor, oque é?


DNS,oque é?

DNS convertem solicitações de nomes em endereços IP, controlando qual servidor um usuário final
alcançará quando digitar um nome de domínio no navegador da web


HTTP, oque é?

HTTP é um protocolo ou conjunto de regras de comunicação para
comunicação entre cliente e servidor


DHCP, oque é?

Trata-se de uma ferramenta que permite aos computadores obter
um endereço IP automaticamente na rede


O que é um servidor FTP?

Um servidor FTP, é a sigla para File Transfer Protocol(e em português é protocolo de transferência de dados).
onde ele ajuda na transferencia de dados para diferentes dispositivos.


Quais são suas principais funcionalidades em uma rede?

Serve como um protocolo de rede padrão usado para a transferência de arquivos de um host para outro por meio de uma rede baseada em TCP.


Por que o FTP foi amplamente utilizado no passado?

Foi por causa da facilidade da troca de arquivos entre computadores diferentes,
na epoca onde as coisas não era tão faceis como hoje em dia.


Vantagens do FTP

Umas principais vantagens do FTP é a trasnferencia simultanea de varios arquivos,
não ter necessidade de um conhecimento profundo da area e os arquivos que foram upados, sempre estarão salvos no servidor


Desvantagens do FTP

as Desvantagens são: manutenção, os servidores, para manter seguros,
precisa de um grande conhecimento na area e tempo, a transferencia tambem pode falhar
e segurança, o FTP nem sempre é seguro, podendo ter falhas.


diferenças entre um servidor FTP e serviços de armazenamento em nuvem

Umas principais seriam a segurança, com o avanço da tecnologia, a melhoria em como aumentar a segurança dos arquivos,
aumentou bastante, o custo para manter os servidores, o FTP é bem economico,
comparado com os armazenamento a nuvem, tendo que pagar para usar ou aumentar o quanto que pode armazenar e a usabilidade,
o ftp é muito antigo e podendo ter um conhecimento na area para poder mexer melhor num servidor ftp,
algo difere para um armazenamento na nuvem hoje em dia, onde é mais amigavel a interface

Servidor de Email

O que é um servidor de e-mail?

É um local de receber e enviar email


quais são suas principais funções em uma rede?

Ele gerencia os email recebidos e envia email, alem de proteger os email, para ninguem ver alem de você e o rementente


O que é SMTP?

Um tipo de protocolo especifico para receber e enviar email


Como funciona SMTP?

esse protocolo é para transmitir dados entre maquinas, independente do hardware e software,
facilitando a comunicação entre as pessoas, o envio do email, executa um programa chamado MTA,
que verifica o dominio do destinatario e se for diferente do rementente, procura o ip dele, para enviar


o que é o POP3?

esse protocolo, é de comunicação utilizado para gerenciar e recuperar email de um servidor


como funciona?

O POP3 normalmente baixa as mensagens para o dispositivo do usuário, permitindo que o usuário acesse os e-mails offline e
Por padrão, após o download das mensagens, elas são deletadas do servidor,
embora muitos clientes de e-mail ofereçam a opção de manter uma cópia no servidor.
