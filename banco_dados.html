banco de dados

o que é um banco de dados?

é uma coleção de informações relacionadas, armazenadas digitalmente e
organizadas em um sistema de computador.


o que é um banco de dados relacional?

É um sistema onde os dados são armazenados em tabelas e relações, com o objetivo de facilitar a compreensão.


que é um banco de dados não relacional?

Um sistema de armazenamento onde não se organiza em tabelas, podendo tambem colocar mais coisas do que simples
dados, como imagem e videos, sendo mais flexiveis.


o que é um diagrama condeitual DER?

u modelo de entendida e relacionamento, usado para poder conversar com cliente que não sabem de programação.


o que é um diagrama lógico MER?

Uma verção mais detalhada do DER, mas, voltado para os programadores da area, para poder programar o modelo fisico
